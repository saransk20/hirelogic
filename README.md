# HireLogic

HireLogic is an AI-powered resume analysis and hiring assistance platform built to simplify and enhance the recruitment process. The application helps users upload resumes, manage candidate profiles, and streamline resume evaluation through a modern web interface.

## Features

- Resume upload and management
- Candidate dashboard
- Resume preview and analysis
- Modern responsive UI
- Authentication support
- Organized component-based architecture
- Built using Next.js and TypeScript

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Clerk Authentication
